Sales Analysis Dashboard using Excel
Project Overview

The Sales Analysis Dashboard is an interactive business intelligence solution developed in Microsoft Excel to analyze sales performance, customer spending behavior, product revenue, seasonal trends, and delivery efficiency. The dashboard provides decision-makers with a centralized view of key business metrics through dynamic charts, KPIs, and slicers.

The primary objective of this project is to transform raw sales data into meaningful insights that help stakeholders monitor performance, identify trends, and make data-driven decisions.

Business Problem

Organizations often store large amounts of sales data, making it difficult to identify:

Which cities generate the highest revenue
Which product categories perform best
Monthly sales trends and seasonal demand
Customer spending patterns
Delivery performance efficiency
Revenue contribution by occasions and festivals

This dashboard solves these challenges by presenting key insights in a visual and interactive format.

Dataset Description

The dataset contains sales transaction information including:

Column Name	Description
Order Date	Date when the order was placed
Delivery Date	Date when the order was delivered
Product Name	Name of the product sold
Category	Product category
Location	Customer city/location
Revenue	Revenue generated from the sale
Occasion	Festival or event associated with purchase
Customer Information	Customer purchase details
